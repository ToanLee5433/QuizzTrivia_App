# Background Music Files

This folder contains background music for the multiplayer quiz game.

## Required Music Files

### 1. lobby-music.mp3
- **Usage**: Plays in the room lobby while waiting for players
- **Type**: Calm, welcoming background music
- **Duration**: 2-3 minutes (loops)
- **Volume**: Low (0.3-0.4)
- **Mood**: Relaxed, friendly, anticipation

### 2. game-music.mp3
- **Usage**: Plays during active quiz gameplay
- **Type**: Upbeat, energetic, tension-building
- **Duration**: 2-3 minutes (loops)
- **Volume**: Medium (0.4-0.5)
- **Mood**: Exciting, focused, competitive

### 3. victory-music.mp3
- **Usage**: Plays on final podium/results screen
- **Type**: Celebratory, triumphant
- **Duration**: 1-2 minutes
- **Volume**: Medium (0.5-0.6)
- **Mood**: Happy, accomplished, rewarding

## How to Add Music

1. Find royalty-free music from sources like:
   - [Incompetech](https://incompetech.com/)
   - [Free Music Archive](https://freemusicarchive.org/)
   - [YouTube Audio Library](https://www.youtube.com/audiolibrary)
   - [Bensound](https://www.bensound.com/)

2. Convert to MP3 format (128-192 kbps recommended)

3. Rename files to match the names above

4. Place files in this directory
