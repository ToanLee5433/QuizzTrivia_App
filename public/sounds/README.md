# Sound Files

Place the following sound files in this directory:

## Game Sound Effects
- `correct.mp3` - Sound when answer is correct
- `wrong.mp3` - Sound when answer is wrong  
- `countdown.mp3` - Countdown sound effect
- `game-start.mp3` - Game starting sound
- `tick.mp3` - Timer tick sound
- `transition.mp3` - Transition between questions

## Modern Multiplayer Music (New)
- `victory.mp3` - Celebratory music for final podium (30-60s)
- `chill.mp3` - Calm ambient background music (loop-able, BPM: 80-100)
- `energetic.mp3` - Upbeat motivating music (loop-able, BPM: 120-140)
- `intense.mp3` - Fast-paced exciting music (loop-able, BPM: 140-170)

## Sound Sources

You can get free sound effects from:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/
- https://pixabay.com/music/ (for music tracks)
- https://incompetech.com/ (royalty-free music)

## Format
- MP3 format
- Keep file sizes under 100KB for sound effects, under 2MB for music
- 44.1kHz or 48kHz sample rate recommended
- Ensure all files are royalty-free for commercial use
